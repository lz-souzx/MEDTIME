MEDTime — Gerenciador de Medicamentos

Descrição:
O MEDTime é um aplicativo desenvolvido em Kotlin no Android Studio para registrar medicamentos, horários, doses e demais informações relacionadas ao tratamento do usuário. Todo o projeto foi construído com interface gerada automaticamente pelo Android Studio, sem uso de XML.

Tecnologias:
Kotlin, Android Studio, SQLite.

Funcionalidades:

- Cadastro de medicamentos;
- Edição e exclusão;
- Lista de medicamentos salvos;
- Armazenamento local via SQLite.

Como executar:

- Clone o repositorio;
- Abra no Android Studio;
- Execute em um dispositivo ou emulador Android.
